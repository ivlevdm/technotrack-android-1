**Профиль человека:**
* фото
* имя, фамилия / никнейм
* контакты
* о себе
* кнопка написать сообщение
* опыт:
  + на чем играет 
  + сколько 
  + тип образования?
* предлагаемые события:
  + события созданные этим человеком
  + события, в которых он участвует, но ищет еще людей (репост) ?
* друзья?
* отзывы с ивентов
 
**Возможные переходы:** 
* назад
* на список друзей?
* сообщение
* событие
* профиль (тыкнуть на написавшего отзыв)
